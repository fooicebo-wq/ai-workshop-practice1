# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 專案說明

這是一個建材採購 / 室內設計案件的**檔案整理工作區**。  
主要任務是讀取各種格式的原始檔，整理成乾淨的 Markdown 文件，不異動原始資料。

---

## 資料整理規則

### 1. 先讀檔，再整理

- 整理任何資料夾前，**必須先完整讀取所有檔案內容**，確認內容後再動作。
- 支援的格式：`.docx`、`.xlsx`、`.xls`、`.csv`、`.txt`、`.md`、`.html`、`.jpg`、`.png`（圖片用視覺辨識）。
- 讀取順序：文字檔 → 試算表 → Word → 圖片（OCR 最後）。

### 2. 不動原始檔

- **嚴禁修改、移動、重新命名任何原始檔案**。
- 整理成果一律以**新建 `.md` 檔案**的方式輸出，存放於原資料夾根目錄。
- 若需要分類，建立新子資料夾，原始檔維持原位。

### 3. 刪除前先確認

- **任何刪除動作，執行前必須列出待刪清單，等待使用者明確同意後才執行**。
- 不確定是否與業務相關的檔案，標記為「待確認」，不自行判斷刪除。
- 範例格式：
  ```
  以下檔案準備刪除，請確認：
  - 購物清單.txt（理由：與採購業務無關）
  確認刪除請回覆「確認」，取消請回覆「取消」。
  ```

## 輸出格式規範

### MD 檔命名
- 總覽檔：`{案件名稱}總覽.md`
- 單一主題：`{主題}.md`

### MD 結構
1. 標題 + 整理日期 + 來源說明
2. 目錄（超過 4 個章節時加入）
3. 各分類內容，優先使用**表格**呈現數據
4. 圖片檔單獨列出（說明來源與內容摘要）
5. 待確認項目獨立一節

### 表格優先
- 報價資料 → 表格（品項 / 規格 / 單價 / 備註）
- 待辦事項 → 表格（# / 工作 / 負責人）
- 廠商清單 → 表格（廠商 / 類別 / 聯絡人 / 電話）

## 常見任務流程

### 整理新資料夾
1. `列出所有檔案` → 確認格式與數量
2. `逐一讀取內容` → 摘要每個檔案
3. `建立總覽 MD` → 依分類整理
4. `回報結果` → 說明整理了哪些、有無待確認項目

### 新增報價資料
1. 讀取新報價檔案
2. 判斷是「新廠商」還是「既有廠商更新」
3. 新廠商 → 在總覽 MD 新增章節
4. 既有廠商 → 找到對應章節更新，並說明改了哪裡
5. 更新總覽末尾的「廠商比較表」

### 讀取圖片
- 手寫文件、名片、報價單照片 → 用視覺辨識讀取
- 說明辨識信心度（清晰 / 部分模糊 / 無法辨識）

## 注意事項

- 購物清單、個人備忘等與業務無關的檔案 → 標記「待確認」，不刪除
- `.dwg`（AutoCAD）格式無法直接讀取，告知使用者需另行開啟
- Excel 試算表優先使用 COM 自動化或 pandas 讀取，保留原始數值
- 所有 MD 檔末尾加上：`*整理工具：Claude Code｜整理日期：YYYY-MM-DD*`
