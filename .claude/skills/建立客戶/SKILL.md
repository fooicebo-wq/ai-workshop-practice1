# 建立客戶資料

將名片或口述的客戶資訊，整理成標準化 Markdown 檔案，存入客戶資料夾。

---

## 執行流程

### Step 1：取得資訊來源

**來源 A：自動掃描目錄找名片（優先執行）**

若使用者說「從目錄找名片」或「掃描目錄」，執行以下步驟：
1. 搜尋目前工作目錄（含子資料夾）所有圖片檔：`.jpg`、`.jpeg`、`.png`
2. 逐一用視覺辨識判斷是否為名片（有公司名稱、電話、地址等聯絡資訊）
3. 列出找到的名片清單，告知使用者：
   > 「找到以下名片，是否全部建立客戶資料？」
4. 使用者確認後，逐張辨識並寫入 `客戶.md`（已存在的客戶略過）

**來源 B：使用者直接提供名片圖片**
- 用視覺辨識讀取圖片中所有文字
- 自動判斷正面（公司/Logo）與背面（聯絡人資訊）
- 辨識信心度低的欄位標記為「（待確認）」
- 若正反兩面都有提供，合併為一份資料

**來源 C：使用者口述**
- 從對話中擷取所有提到的聯絡資訊
- 若完全沒有客戶名稱（姓名或公司名稱皆無），**立即停止並告知使用者**：
  > 「找不到客戶名稱，請提供客戶的姓名或公司名稱後再試一次。」
- 有名稱但缺少其他欄位，正常建立，缺漏欄位留空即可

### Step 2：整理欄位

盡量填寫以下欄位，沒有的留空即可，不需強迫詢問：

| 欄位 | 說明 |
|------|------|
| 姓名 | 中文名 + 英文名（若有） |
| 職稱 | 例：設計總監、業務經理 |
| 公司 | 全名 + 英文名（若有） |
| 統一編號 | 八碼數字 |
| 地址 | 公司地址 |
| 電話 | 市話（含區碼） |
| 手機 | 行動電話 |
| Email | 電子信箱 |
| 網址 | 公司官方網站（見下方自動查詢流程） |
| 業務範圍 | 名片上標示的服務項目 |
| 備註 | 其他資訊（LINE ID、QR code 說明等） |

### Step 2.5：上網查詢公司官方網址

**不論名片上有沒有網址，都執行以下查詢流程：**

① **名片上已有網址** → 以名片網址為主，但仍用 WebSearch 確認該網址是否有效

② **名片上無網址** → 立即使用 WebSearch 工具主動上網搜尋：

```
搜尋關鍵字："{公司全名}" 官網
或："{公司全名}" 網站
或："{公司名稱關鍵字}" site:.com.tw
```

搜尋後從結果中找出最符合的官方網址：
- ✅ **採用**：網址含公司名稱、有公司介紹頁、域名與公司相符（.com.tw / .com / .tw）
- ❌ **不採用**：黃頁、104人力銀行、商業司查詢、FB 粉絲頁、Google Maps

找到 → 填入網址欄，備註加上「（網址由搜尋取得）」
找不到 → 網址欄留空，備註「網址：搜尋後未找到官網」

### Step 3：寫入統一客戶檔案

所有客戶資料統一存放於專案目錄下的 **`客戶.md`**，絕對不建立個別客戶檔案。

**檔案位置**：目前工作目錄（與 CLAUDE.md 同層）的 `客戶.md`

#### 處理邏輯（按順序判斷）

**① 先確認 `客戶.md` 是否存在**

- **不存在** → 立即建立，寫入標題後加入第一筆客戶資料：
  ```
  # 客戶資料總覽
  整理日期：{今日日期}
  ---
  {客戶區塊}
  ```
- **已存在** → 讀取全部內容，進入下一步判斷

**② 確認此客戶是否已在檔案中**（搜尋姓名或公司名）

- **已有此客戶** → 不重複新增，告知使用者「此客戶已存在」，並詢問是否要更新資料
- **尚無此客戶** → 在檔案末尾附加新的客戶區塊（保留既有內容不動）

#### 每筆客戶區塊格式

```markdown
## {姓名}（{英文名}）｜{公司簡稱}

- **建立日期**：{今日日期 YYYY-MM-DD}
- **來源**：{名片掃描 / 口述}

| 欄位 | 內容 |
|------|------|
| 姓名 | {姓名}（{英文名}） |
| 職稱 | {職稱} |
| 公司 | {公司全名} |
| 統一編號 | {統編} |
| 地址 | {地址} |
| 電話 | {電話} |
| 手機 | {手機} |
| Email | {email} |
| 網址 | {公司網址} |

**業務範圍**：{服務項目（以「・」分隔）}

**備註**：{其他資訊，若無則省略此行}

---
```

若某欄位無資料，該列留空或省略，保持整潔。

### Step 4：回報結果

告知使用者：
- ✅ 已寫入 `客戶.md`（新增 或 更新）
- 辨識到哪些欄位（以條列方式呈現）
- 哪些欄位辨識不清或待確認（若有）

---

## 辨識注意事項

- 名片上的電話格式多樣（04-1234-5678、0912345678、+886-4-...），統一整理後輸出
- QR Code 內容無法直接讀取，備註為「名片含 QR Code（內容需另行掃描）」
- 若名片有 LINE ID，記錄於備註欄
- 辨識信心度：清晰／部分模糊／無法辨識，模糊處加「（待確認）」

---
